# Практическая работа 6: OpenCL для CPU и GPU

## Описание
Этот проект демонстрирует основы OpenCL и я реализовала две задачи:
1. Сложение двух массивов (Task1)
2. Умножение двух матриц (Task2)


## Структура проекта
PracticeWork6/
│
├─ Task1/
│   ├─ main.cpp     # Программа для сложения массивов
│   └─ kernel.cl    # Ядро OpenCL для сложения массивов
│
└─ Task2/
    ├─ main.cpp     # Программа для умножения матриц
    └─ kernel.cl    # Ядро OpenCL для умножения матриц


## Результат
Программа измеряет время выполнения ядра на выбранном устройстве, чтобы можно было сравнить производительность CPU и GPU.

Task1 (сложение массивов)
Запуск на CPU:
Время выполнения: 48.7 мс
C[0] = 3, C[N-1] = 3

Запуск на GPU:
Время выполнения: 2.1 мс
C[0] = 3, C[N-1] = 3

Task2 (умножение матриц 512x512)
Запуск на CPU:
Время выполнения: 432.5 мс
C[0] = 1024, C[N*K-1] = 1024

Запуск на GPU:
Время выполнения: 21.3 мс
C[0] = 1024, C[N*K-1] = 1024

Значения времени зависят от мощности процессора и видеокарты. 

## Сравнение времени выполнения на CPU и GPU
Сложение массивов	CPU	= 48.7мс
Сложение массивов	GPU	= 2.1мс
Умножение матриц 512x512	CPU	= 432.5мс
Умножение матриц 512x512	GPU	= 21.3мс

Вывод:

GPU показывает значительное ускорение по сравнению с CPU, особенно на больших данных.

Для векторных операций ускорение меньше, но всё равно
