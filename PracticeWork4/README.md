Что делает проект

Генерирует массив случайных чисел.

Считает сумму элементов массива двумя способами:

с использованием только глобальной памяти,

с использованием shared memory для ускорения.

Измеряет время выполнения каждого метода для разных размеров массива.

Сохраняет результаты в CSV-файл для построения графиков.


Выводы:

Использование shared memory ускоряет редукцию почти в 2–3 раза, потому что потоки работают с быстрой памятью внутри блока и меньше обращаются к медленной глобальной памяти.

Чем больше массив, тем более заметна разница между глобальной и shared памятью.

Глобальная память подходит для хранения больших массивов, но для промежуточных вычислений всегда лучше использовать shared memory.
